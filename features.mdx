## User Roles and Permissions

Define roles to manage access securely. Assign permissions like view-only, edit, or admin.

<Callout kind="alert">
  Always review permissions before sharing schedules to prevent unauthorized changes.
</Callout>

| Role       | View | Edit | Share | Admin |
|------------|------|------|-------|-------|
| Student    | ✅   | ❌   | ❌    | ❌    |
| Teacher    | ✅   | ✅   | ✅    | ❌    |
| Coordinator| ✅   | ✅   | ✅    | ✅    |

Use the roles panel to assign users:

```javascript
// Example API call to update roles
const updateRole = async (userId, role) => {
  const response = await fetch(`https://api.example.com/schedules/123/roles`, {
    method: 'POST',
    headers: { 'Authorization': `Bearer ${YOUR_TOKEN}` },
    body: JSON.stringify({ userId, role: 'teacher' })
  });
  return response.json();
};
```

## Viewing and Exporting Schedules

View schedules in calendar, list, or timeline modes. Export to PDF, ICS, or CSV for sharing.

<CodeGroup tabs="JavaScript,Python">
  ```javascript
  // Export to ICS format
  const exportICS = async (scheduleId) => {
    const response = await fetch(`https://api.example.com/schedules/${scheduleId}/export?format=ics`, {
      headers: { 'Authorization': `Bearer ${YOUR_TOKEN}` }
    });
    const blob = await response.blob();
    const url = URL.createObjectURL(blob);
    const a = document.createElement('a');
    a.href = url;
    a.download = 'schedule.ics';
    a.click();
  };
  ```
  ```python
  # Export to CSV
  import requests

  headers = {'Authorization': f'Bearer {YOUR_TOKEN}'}
  response = requests.get('https://api.example.com/schedules/123/export?format=csv', headers=headers)
  with open('schedule.csv', 'wb') as f:
      f.write(response.content)
  ```
</CodeGroup>

## Integration with Calendars

Sync schedules with Google Calendar, Outlook, or iCal. Push updates automatically.

<Expandable title="Setup Google Calendar Integration" default-open="true">
  1. Navigate to **Integrations** > **Calendars**.

  2. Click **Connect Google**.

  3. Authorize and select calendars to sync.

  Changes in Расписание КГУ update your calendar in real-time.
</Expandable>
