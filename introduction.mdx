## Overview

Расписание КГУ provides a comprehensive system for managing schedules at Kyrgyz State University (КГУ). You create, edit, and share timetables efficiently, eliminating manual spreadsheets and errors. Designed for administrators, teachers, and students, it centralizes all scheduling needs in one intuitive platform.

<Callout kind="info">
  Access the dashboard at `https://dashboard.raspkgu.com` after signing up.
</Callout>

## Quick Start

Get up and running in minutes.

<Steps>
  <Step title="Sign Up" icon="user-plus">
    Visit `https://dashboard.raspkgu.com/signup` and create your admin account with your КГУ email.
  </Step>
  <Step title="Create First Schedule">
    Navigate to Schedules > New Schedule. Add groups, subjects, and rooms.
  </Step>
  <Step title="Share Schedule">
    Click Share and generate a public link. Copy the embed code for your university portal.
  </Step>
</Steps>

For API integration, use this example to fetch schedules programmatically:

<CodeGroup tabs="JavaScript,Python">
  ```javascript
  const response = await fetch('https://api.example.com/v1/schedules/123', {
    headers: {
      'Authorization': `Bearer ${YOUR_API_KEY}`
    }
  });
  const schedule = await response.json();
  console.log(schedule);
  ```
  ```python
  import requests

  headers = {'Authorization': f'Bearer {YOUR_API_KEY}'}
  response = requests.get('https://api.example.com/v1/schedules/123', headers=headers)
  schedule = response.json()
  print(schedule)
  ```
</CodeGroup>
